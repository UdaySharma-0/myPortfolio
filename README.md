# 🌐 My Portfolio

A personal portfolio website showcasing my projects, skills, and achievements in web development and software engineering. Built with a focus on clean design, responsive layout, and efficient backend integration.

🔗 **Live Demo:** [udaysharmaportfolio.netlify.app](https://udaysharmaportfolio.netlify.app/)

---

## 📂 Project Structure

- **index.html / index1.html** → Main portfolio pages  
- **style.css** → Custom styling and layout  
- **contact.php** → Contact form handling  
- **my_database.sql** → Database schema for storing contact form submissions  
- **Images/** → Assets and visuals used across the site  
- **UDAY SHARMA webdeveloper.pdf** → Resume/CV  

---

## 🚀 Features

- Responsive design for desktop and mobile  
- Interactive contact form with PHP backend  
- Database integration for form submissions  
- Modern styling with CSS  
- Easy deployment on Netlify/Vercel  

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Deployment:** Netlify, Vercel
